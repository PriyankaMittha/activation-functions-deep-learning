# 🔥 Activation Functions in Deep Learning

## 📌 Overview

This project demonstrates the implementation of various activation functions used in deep learning along with their derivatives and graphical visualizations.

Activation functions play a crucial role in neural networks by introducing non-linearity, allowing models to learn complex patterns.

---

## ⚙️ Activation Functions Covered

* Sigmoid
* ReLU (Rectified Linear Unit)
* Tanh
* Leaky ReLU
* Parametric ReLU (PReLU)
* Softmax

---

## 📊 Visualizations

Each activation function is plotted using Matplotlib to understand its behavior and output range.

---

## 🧠 Key Concepts

* Non-linearity in neural networks
* Vanishing gradient problem
* Dead neuron problem
* Probability distribution using Softmax

## 🎯 Key Takeaways

* ReLU is the most widely used activation function
* Sigmoid and Tanh suffer from vanishing gradient
* Leaky ReLU helps overcome dead neuron problem
* Softmax is used for multi-class classification
